Wikidata


  • Es un proyecto de la fundación Wikimedia, una organización sin ánimo de lucro cuyo objetivo es llevar el conocimiento a todos los habitantes del planeta; Wikipedia es su proyecto más conocido, Wikidata apareció en 2012.


  • Wikidata es una base de datos colaborativa, libre y con información enlazada y estructurada. Su principal ventaja es que ofrece datos enlazados, lo que permite relacionarlos con otros conjuntos de datos y hacer búsquedas de información con sentido. Aquí y aquí explican bien su utilidad.

  • glitter: This package aims at writing and sending SPARQL queries without advanced knowledge of the SPARQL language syntax. It makes the exploration and use of Linked Open Data (Wikidata in particular) easier for those who do not know SPARQL well.

Obtención de los datos

  • Procedentes de Wikidata (usando R y el paquete wikifacts)

  • Primero obtuve el listado de premios literarios existentes en Wikidata con la siguiente query:

query_premios <- '#title:premios literarios en wikidata
SELECT  ?award ?awardLabel ?pais_premioLabel ?iconoLabel ?web_premioLabel ?anyo_creacionLabel ?da_premioLabel ?prizeLabel WHERE {
  ?award wdt:P31 wd:Q378427 .
  OPTIONAL {?award wdt:P17    ?pais_premio . }
  OPTIONAL {?award wdt:P571  ?anyo_creacion .  }
  OPTIONAL {?award wdt:P856   ?web_premio . }
  OPTIONAL {?award wdt:P1027  ?da_premio . }
  OPTIONAL {?award wdt:P2121  ?prize . }
  OPTIONAL {?award wdt:P2910  ?icono . }

  SERVICE wikibase:label { bd:serviceParam wikibase:language "es, en, fr, it, ca". }
}'

wikifacts::wiki_query(query_premios)


  • La query se realizó el 13 de octubre de 2023, obteniéndose datos de un total de 4.159 premios literarios de 106 países distintos.
  • Posteriormente implementé (con un loop) la siguiente consulta, obteniendo datos de los galardonados de cada uno de los premios literarios
query_premiados <- '#title: premiados de un premio literario (en este caso el Cervantes: wd:Q81466)
SELECT DISTINCT ?person ?personLabel ?genderLabel ?year_nac ?educated ?educatedLabel ?nacionalidadLabel ?award ?awardLabel  ?year_premio ?lugar_nacLabel ?countryLabel ?fecha_nacimiento ?fecha_muerte ?_image ?_coordinates ?signature WHERE {
  ?person p:P166 ?statement.
  ?statement ps:P166 ?award.
  ?award wdt:P279?/wdt:P31?  wd:Q81466.

  ?person wdt:P21 ?gender.

  OPTIONAL {
    ?statement pq:P585 ?date.
    BIND(YEAR(?date) AS ?year_premio)
  }
  OPTIONAL { ?person wdt:P569 ?fecha_nacimiento. }
  OPTIONAL {
    ?person wdt:P569 ?fecha_nacimiento.
    BIND(YEAR(?fecha_nacimiento) AS ?year_nac)
  }

  OPTIONAL { ?person wdt:P570 ?fecha_muerte. }
  OPTIONAL { ?person wdt:P18 ?_image. }
  OPTIONAL { ?person wdt:P101 ?ocupacion. }
  OPTIONAL { ?person wdt:P69 ?educated. }
  OPTIONAL { ?person wdt:P27 ?nacionalidad.  }

  OPTIONAL {
    ?person wdt:P19 ?lugar_nac.
    ?lugar_nac wdt:P625 ?_coordinates.
    ?lugar_nac wdt:P17 ?country.
  }
  SERVICE wikibase:label { bd:serviceParam wikibase:language "es, en, fr, it, ca". }
}
ORDER BY DESC (?year) (?awardLabel)

wikifacts::wiki_query(query_premiados)
  • La consulta se realizó el 13 de octubre de 2023, obteniéndose datos de un total de 48.261 premios totales otorgados, correspondientes a 30.235 escritores galardonados con al menos un premio literario.

Curación de datos


  • Se detectaron algunos errores y se corrigieron (p.ej: Juan Rulfo no ganó el Alfaguara de Novela. Boris Izaguirre no ganó el premio Planeta, sólo fue finalista)


  • Algunos premios son compartidos por varios países (selecciono solo el primer país)


  • Algunos escritores, concretamente 12, tenían asignados varios géneros; por ejemplo Jaye Simpson tenía 3 géneros, entre ellos two-spirit
